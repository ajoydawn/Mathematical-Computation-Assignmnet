📌 Assignment: Linear Systems & Signal Analysis

This repository contains solutions to two computational physics problems:

🔹 1. Gauss-elimination

A system of linear equations is solved using Gauss-elimination to obtain the solution.

🔹 2. Gaussian Pulse Analysis

A Gaussian signal is analyzed using:

Fourier Transform (DFT & FFT) to study frequency behavior


Analytical comparison with the exact transform


Parameter effects such as width σ and time shift $t_0$


Wavelet Transform (Morlet) to examine time–scale localization


📊 Key Insights


Wider pulses in time lead to narrower frequency spectra


Time shifts affect only phase, not magnitude


Wavelet transform provides localized time–frequency information


🛠️ Tools Used


Python (NumPy, Matplotlib)


SciPy (FFT)


PyWavelets (CWT)


📎 Notebook

The full implementation and results are available in the Jupyter Notebook included in this repository.
